# MARKDOWN

## El lenguaje de marcado

**Markdown** está diseñado para agregar formato, imágenes, vínculos, etc, ... *con una gran facilidad*, se puede usar en:

- Comentarios
- Macros
- Firmas de agentes
- Contenido dinámico
- Bloques de contenido

### Por lo que en nuestros documentos nos seria de gran ayuda ya que, nos permite comentar el contenido del documento y así saber cual es la función de cada elemento contenido en el proyecto.

  

Este lenguaje permite dar formato a textos empleando una serie de caracteres de forma especial, se creó con la intención de sustituir a las páginas web anteriores por su *rapidez* y *sencillez*

  

## Ventajas de usar Markdown

1. sencillez
2. portabilidad
3. flexibilidad
4. compatibilidad

> **Consejo:** Puedes colocar 3 "~" para colocar código sin que lo interprete como:

~~~
#Markdown
##El lenguaje de marcado

~~~
